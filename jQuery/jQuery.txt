＊jQuery＊

jQueryは、JavaScriptのコードを
使いやすく簡単にまとめたコード集(ライブラリ)です。

jQueryでできること。
１．スライドショーを作ることができる
２．クリックすると画像が大きく表示される
３．文字にマウスを乗せると色が変わる
他にもWebページに様々な動きを加えることができる


JavaScriptファイルにjQueryコードを記述

$(function(){
    //ここにjQueryを書いていく
});


基本的な記述法

$(function(){
    $("セレクタ").メソッド(引数);
});

セレクタは（id）（class）（子）などを記述し、対象となる部分を指定する
メソッドは指定したセレクタに対して、何の処理を実行するかを指定する
引数はメソッドの動作を制御する


セレクタとは             例　　　　　　　　　　　　書き方
1.要素セレクタ　　h1,h2,p,ul,img,form　　　$("h1").css("color","red");
2.任意セレクタ　　.abc, .box, .name        $(".abc").css("color","red");
3.IDセレクタ　　　#abc, #box, #name        $("#abc").css("color","red");

複数のセレクタ
1.グループセレクタ　　　　要素セレクタを(「,」コンマ)で区切る
2.スペースセレクタ　　　　　任意セレクタ、IDセレクタを(「　」スペース)で区切る


重要なメソッド
　メソッド　　　　　　　　　　　用途
   .css()         CSSプロパティを変化させます。
    .on()        イベント発生時に実行する関数を割り当てます。
 .fadeOut()      要素を徐々に薄くして消していきます。(=フェードアウトさせる)
 .fadeIn()       要素を徐々に表示していきます。(=フェードインさせる)
  .text          指定した要素の文字列を取得したり、書き換えたりします。
.slideToggle()   要素をアコーディオンさせます。
 .animate()      CSSプロパティを徐々に変化させます。


表示させるには、まずhtmlで記入する
例　　HTMLファイル　　　　　　　　　JavaScript
  <h1>Hello guys</h1>             $(function(){$("h1").css("color","red");});

          すると画面では「Hello guys」が赤くなった状態で表示される


イベント

基本的な記述法
$(function(){
$("セレクタ１").イベント(function{
 $("セレクタ２").メソッド(引数);});
});

イベント集
　　  種類　　　　　　　　　　　　　用途
　　　click　　　　　　クリックした時に○○の処理をする
　　mouseover　　　　マウスオーバーした時(=マウスをのせた時)に○○の処理をする
　　mouseout　　　　　マウスアウトした時(=マウスを外した時)に○○の処理をする
　　　hover　　　　　マウスオーバーした時(=マウスをのせた時)に○○の処理をする
　　　　　　　　　　　　　　※mouseoverとは、若干の違いがあります
　　　scroll　　　　　　　画面をスクロールした時に○○の処理をする
　　dbclick　　　　　　　　ダブルクリックをした時に○○の処理をする

例                                JavaScript
    HTML                     $(function(){ 
<h1>hello</h1>                $("h1").click(function(){
<p>what`s up</p>               $("p").fadeOut();
                              });
                             });
すると画面ではwhat`s upがフェードアウトして消える。


hover

記述法
$(function(){
 $("セレクタ").hover(
  function(){
   },
  function(){
   });
});


this
thisとはイベント発生時（クリックした時など）に そのセレクタのみ（クリックされたセレクタ）のみに処理を適用する。

記述例
$(function(){
$(“h2”).click(function{
$(this).css(“color”,”red”);　　　＝＞　　　この場合クリックされた"h2"だけ赤くなり、それ以外の"h2"は変わらない。
}); });　　　　　　　　　　　　　　　　　　　＊この(this)がないと"h2"がクリックされた時、全ての"h2"が赤くなる。
